
# Week 2: Object Detection using YOLO

## Objective
To perform object detection using pretrained YOLOv8 model.

## Steps Performed
1. Created Python virtual environment using venv
2. Activated virtual environment
3. Installed ultralytics package using pip
4. Ran YOLOv8 pretrained model

## Command Used
yolo predict model=yolov8n.pt source='https://ultralytics.com/images/bus.jpg'

## Output
The model successfully detected objects such as persons and vehicles in the image.

## Result
Object detection was performed successfully using pretrained YOLO model.
