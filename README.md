# Video Processing using FFmpeg

This repository contains basic Computer Vision preprocessing tasks:

## Tasks Completed

### Task 1: Frame Extraction
Extracted frames from video using FFmpeg.

### Task 2: Video Reconstruction
Converted frames back into video (30 FPS).

### Task 3: Audio-Video Merging
Merged background audio with video.

## Tools Used
- FFmpeg
- YouTube videos
- Pixabay audio

## Learning Outcome
- Understanding video as frames
- Frame rate (FPS)
- Multimedia processing
