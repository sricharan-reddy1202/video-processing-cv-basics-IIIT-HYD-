
# Week 2 Advanced Task

## Objective
Apply object detection on multiple images and generate a video with annotations and audio.

## Steps Performed
1. Used frames extracted in Week 1
2. Applied YOLOv8 object detection on multiple images
3. Generated annotated images
4. Converted images into video using FFmpeg
5. Added background audio to the video

## Tools Used
- Python (venv)
- Ultralytics YOLOv8
- FFmpeg

## Result
Successfully created a video with object detection annotations and background music.
